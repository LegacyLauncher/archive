TL;DR: Открылся этот файл вместо лаунчера? Запускай лаунчер джавой!

Привет! Спасибо за установку Legacy Launcher!
Если у тебя открылся архив с этим файлом вместо лаунчера, то тебе нужно:
1. Кликнуть LL правкой кнопкой мыши
2. Выбрать "Свойства"
3. В строчке "Приложение" кликнуть "Изменить..."
4. В появившемся окне выбрать Java
4.1. В появившемся списке нет Java? Установите её с java.com/download
4.2. Установили, но Java всё равно нет в списке?
     Используйте JarFix: https://johann.loefflmann.net/en/software/jarfix/index.html
5. Нажмите "ОК" и закройте окно "Свойства". Готово!

У тебя Linux? Найди в репозитории пакет с Java 8 (он обычно называется openjdk-8-jre, jre8-openjdk)
и установи его с помощью пакетного менеджера (apt, apt-get, pacman, pamac, dnf, yum, portage...)

У тебя macOS? Просто установи Java с java.com/download

======================================
     Продвинутые функции лаунчера
======================================
* Эти аргументы нужно указывать либо в файле tl.bootargs, лежащим рядом с лаунчером
* либо перед параметром -jar в команде запуска в ярлыке или терминале (т.е. java АРГУМЕНТЫ -jar LL.jar)
* Рекомендуется использовать не общий tl.booargs, а для конкретной ОС
* В таком случае, файл будет называться tl-[ОС].bootargs
* Есть поддержка разрядности ОС, в таком случае файл будет называться tl-[ОС]-[РАЗР].bootargs
* вместо [ОС] - windows, linux, osx соответственно
* вместо [РАЗР] - x64 для 64-бит систем и x86 для 32-бит систем

Этот параметр указывает путь к файлу лаунчера.
Нужен только для портативного использования.
-Dtlauncher.bootstrap.targetJar=<путь/к/файлу.jar>

Этот параметр указывает путь к папке с библиотеками лаунчераю
Нужен только для портативного использования
Может использоваться для перемещения файлов лаунчера на другой диск
-Dtlauncher.bootstrap.targetLibFolder=<путь/к/папке>

======================================
* Эти аргументы нужно указывать либо в файле tl.args, лежащим рядом с лаунчером
* либо в качестве аргументов запуска лаунчера (т.е. java -jar LL.jar АРГУМЕНТЫ)

Принудительно указывает путь к папке игры. Отключает смену папки игры через настройки.
Обычно используется для портативного использования
--directory <путь/к/папке>

Принудительно указывает аргументы Java. Отключает смену аргументов Java в лаунчере
--javaargs <аргументы>

Принудительно указывает аргументы запуска игры. Отключает их смену в лаунчере
--margs <аргументы>

Задает путь к файлу конфигурации лаунчера вместо стандартного
--settings <путь/к/файлу>

Задает аккаунт, выбранный в лаунчере по умолчанию
--usernane <никнейм>

Задаёт версию, выбранную в лаунчере по умолчанию
--version <версия>

Принудительно задаёт указанный фон в лаунчере. Отключает смену фона в настройках
--background <путь/к/файлу>
