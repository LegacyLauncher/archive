Если по умолчанию Вы открыли лаунчер с помощью архиватора (WinRar, 7-Zip и другие),
в данном файле предложены решения данной проблемы.

= Простое решение:
-- Скачайте EXE-пакет TLauncher: http://u.to/tlauncher-exe/tdUkBA

= Более сложное решение, которое может решить и другие проблемы:
- Откройте свойства скачанного файла. То есть, кликните правой мышью по TLauncher.jar
(или как он у Вас называется), выберите в появившемся контекстном меню "Свойства",
найдите строку "Приложение", нажмите "Изменить", затем выбрите Java(TM) Platform SE
binary. Если такой программы нет, найти её обычно можно по следующему пути:
"C:\Program Files\Java\", выберите любую папку в этой директории, затем найдите папку
"bin", в ней выберите файл "javaw.exe". Не забудьте отметить галочку "Использовать
данную программу для всех файлов такого типа". Нажмите "ОК", попробуйте запустить.
Однако, у Вас может быть не установлена Java вообще, то есть Вы не смогли найти её
папку. В таком случае, скачайте установочный пакет по следующей ссылке: http://java.com
